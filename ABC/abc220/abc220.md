## C - Long Sequence
https://atcoder.jp/contests/abc220/tasks/abc220_c

<details><summary>クリックで解説を開く</summary>

<br>

今回の問題・制約では気にしなくても良いことなのだが、もし問題が"和が初めて $`X`$ **以上になる**のは"で、かつ制約が $`0 ≤ A_i ≤ 10^9`$ の場合は注意が必要。<br>
$`A`$ の最後の項が0で、$`A`$ を1周する前に和が $`X`$ になっている可能性を考えなければいけないからである。<br>
この場合は、何周するのかを求める際、`X // s` ではなく、`(X + s - 1) // s - 1` として(つまり切り上げてから1周減らして)、和が $`X`$ 以上になるまで $`A_i`$ を足していけば良い(下記参照)。

```python
N = int(input())
A = list(map(int, input().split()))
X = int(input())

s = sum(A)
cycle = (X + s - 1) // s - 1
ans = N * cycle
X -= s * cycle
while X > 0:
    X -= A[ans % N]
    ans += 1

print(ans)
```

### キーワード
- 周期性
