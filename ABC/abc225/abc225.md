## A - Distinct Strings
https://atcoder.jp/contests/abc225/tasks/abc225_a

<details><summary>クリックで解説を開く</summary>

<br>

公式の解答にある通り、$`S`$ に含まれる文字の種類が3種類なら6通り、2種類なら3通り、1種類なら1通り。

$`S`$ に含まれる文字の種類数は、$`S`$ をset型(集合型)にしたものを用意し、その要素数を見れば良い。("abc225_a 1.py")

"abc225_a 2.py"では `itertools.permutations` を使い、$`S`$ の文字を並び替えてできるものを全探索してset型(集合型)に入れ、その要素数を出力している。$`S`$ の文字数を $`n`$ として(今回は $`n=3`$)、計算量は $`O(n!)`$ だと思う。

"abc225_a 3.py"では、$`n`$ が大きくなっても良い解法。<br>
例えば入力例 1では、`a` が2文字、`b` が1文字。最初 `a` の場所を決めようとすると、3文字の中から2文字を選ぶので $`_3 \mathrm{C}_2 = 3`$ 通り。次に`b` は、残りの1文字の中から1文字を選ぶので $`_1 \mathrm{C}_1 = 1`$ 通り、という感じ。計算量は $`O(N)`$ だと思う。

### キーワード
- 文字列の処理
- 場合の数

## C - Calendar Validator
https://atcoder.jp/contests/abc225/tasks/abc225_c

<details><summary>クリックで解説を開く</summary>

<br>

$`1 ≤ j ≤ 7`$ というのが、7で割った余りである0〜6と噛み合わず気持ち悪いので、-1することで $`0 ≤ j ≤ 6`$ にして合わせる。

$`B_{1,1}`$ から $`i, j`$ を求め、$`B`$ のほかの要素がこの $`i, j`$ に噛み合っているかを確認する。

入出力例にはないが、例えば
```
3 2
7 8
14 15
21 22
```
という入力は、$`1 ≤ j ≤ 7`$ に反している(1行の中で週をまたいでいる)ためアウト。出力の答えは
```
No
```
となる。
